Objectifs du jeu

Votre mission est de recouvrir toutes les cases cibles de la grille en utilisant les briques disponibles dans la pioche.
Tentez de le faire en le moins de tours possible !


Déroulement du jeu

La grille est composée de cases vides et de cases cibles (indiquées par des motifs hachurés) : les cases cibles sont celles que vous devez recouvrir.
La pioche contient 4 briques disponibles : chaque brique a des dimensions et doit être placée correctement pour recouvrir les cases cibles.


Comment jouer ?

A chaque tour :

  • Choisissez une brique dans la pioche.
  • Décidez de votre action :
    -> Placer la brique sur la grille si elle peut recouvrir des cases cibles encore disponibles.
    -> Défausser la brique pour la remplacer par une nouvelle brique tirée aléatoirement.

Placement des briques :

  • Une brique peut être placée uniquement si toutes ses cases recouvrent des cases cibles disponibles.
    Par exemple, une brique de 2×3 nécessite 6 cases cibles libres.

Défausse

  • Si une brique ne convient pas, vous pouvez la défausser.
    Elle sera remplacée par une nouvelle brique tirée dans la base de données.

Fin du jeu et score

  • Le jeu se termine lorsque toutes les cases cibles sont recouvertes.
  • Votre score correspond au nombre de tours nécessaires pour atteindre cet objectif.
  • Objectif : Réalisez le score le plus bas possible !
